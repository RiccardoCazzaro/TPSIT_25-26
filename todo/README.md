# Todo List - Stile Google Keep

Cazzaro Riccardo 5IE

## Descrizione
Applicazione Todo ispirata a Google Keep.<br>Permette di aggiungere, modificare, completare ed eliminare note Todo  

- Tutti i Todo sono visualizzati come card verticali
- Tap sul testo per modificarlo direttamente (TextField)
- Checkbox per segnare come completato
- FloatingActionButton per aggiungere nuove note

## Scelte di sviluppo
- **Card + Column invece di ListView.builder:** stile simile a Google Keep.
- **TextField inline per il testo dei Todo** 
- **Checkbox per completamento** 
- **IconButton per eliminare nota**
